# Alma

Alma is a SaaS app that allows students to create and sign up for tasks within their own school community.

## Set up
- Run 'bundle install'
- Run 'rake db:seed'
- Run 'rake db:migrate'

## To run tests
- Run 'rake cucumber'
- Run 'rspec spec'
